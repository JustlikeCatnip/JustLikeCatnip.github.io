  • …rest 参数

箭头函数的简写规则：

  • 当无形参时，只需将普通函数的 function 和 ( ) 省略为一个括号即可
  • 当有一个形参时，可直接只保留一个形参
  • 当有两个及以上参数时，需要完整写为(参数1,参数2,……)的形式
  • 当只有一条语句时，{}也可以不加 return也要去掉，表示函数直接返回这条语句(所有函数写法都适用这条规则)
  • 可以使用 ()和,的方式简写代码块(所有函数写法都适用这条规则)
